# Todo App
Це простий Todo додаток, з повною взаємодією з API та покращеним UX.

## Основні функції

- **Завантаження Todos:**
  Завантаження списку задач з API з обробкою помилок та спінерами під час завантаження.

- **Додавання Todo:**
  - Поле вводу фокусується автоматично.
  - Порожній заголовок показує повідомлення про помилку.
  - Під час відправки на API створюється тимчасовий елемент з loader.
  - Після успішного додавання поле очищується, при помилці текст зберігається.

- **Видалення Todo:**
  - Видалення окремої задачі або всіх виконаних.
  - Loader накладається на видалювану задачу.
  - У разі помилки відображається повідомлення, і Todo залишаються у списку.

- **Тоглінг статусу:**
  - Можливість відмітити задачі як виконані/невиконані.
  - Тоглінг усіх задач одночасно через `toggleAll`.
  - Під час оновлення статусу показується loader.
  - У разі помилки показується повідомлення.

- **Редагування Todo:**
  - Подвійний клік дозволяє редагувати заголовок.
  - Зміни зберігаються по Enter або на Blur, ESC скасовує редагування.
  - Порожній заголовок видаляє Todo.
  - Під час збереження показується loader.
  - У разі помилки відображається повідомлення.

## Покращення UX

- Автоматичний фокус на полях вводу.
- Елементи, які не можна використовувати, приховані або відключені.
- Запобігання дублювання дій (відключення кнопок під час обробки).
- Спінеpи для повідомлення користувача про активність.
- Повідомлення про помилки з автоматичним схованням.

## Технології

- React + TypeScript
- Redux для глобального стану
- CSS Modules для стилізації
- Fetch API для взаємодії з бекендом
